# Dashboard Guide

The dashboard has three main areas so you can control your overlay quickly during a stream.

## Three-column layout

### Left sidebar

Use this area to:
- Add widgets
- Manage scenes
- Save and load presets

### Canvas

This is your stream canvas preview.

You can:
- Click widgets to select them
- Drag and resize to position them
- Build your layout exactly where you want it

### Right sidebar

Use this area to:
- Edit the selected widget settings
- Open the media library
- Trigger sounds from the persistent Sounds panel

## Top navbar

The top bar shows:
- Connection status (Live or Disconnected)
- Your signed-in username and role
- Logout button

If status is Live, your actions are syncing in real time.

## Visible vs hidden widgets

Each widget can be visible or hidden.

- Visible: viewers can see it on stream.
- Hidden: you can still position and configure it, but viewers cannot see it yet.

This is useful for preparing elements before revealing them.

## Scenes

Scenes are different overlay layouts for different stream segments, like gameplay, BRB, or ending.

Switching scenes swaps to that layout immediately.

## Presets

Presets are saved snapshots of your current scene layout.

Use presets when you want to:
- Save a known-good version of a layout
- Restore a previous arrangement quickly
- Jump between recurring stream formats

You can save the current scene, then load it later to replace the current scene widgets instantly.
