# FAQ

## Do my mods need to install anything?

No. Any browser on any device works.

## Will mods break my stream?

No. They control a browser canvas layered over your stream.
They have no access to OBS settings, audio mixer, or keys.

## What happens if the server restarts?

Your widgets and settings are saved automatically. Log back in.

## Can I use this without Docker?

Yes, but Docker is recommended. Developer docs cover manual setup.

## Is my data sent anywhere?

No. Everything stays on your server. Anomalist is self-hosted.

## I forgot my password. How do I reset it?

The owner can reset any user's password from User Management.
If you lost owner access, delete anomalist.db and run setup again.
