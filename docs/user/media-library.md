# Media Library

The Media Library stores files you use in widgets.

## Uploading media

You can upload by:
- Drag and drop
- Clicking the Upload button and selecting a file

## Supported file types

- Images
- Video
- Audio

## Using media in widgets

Common use cases:
- Image widget for logos, frames, or graphics
- Soundboard widget for audio triggers

After upload, select an item from the library to apply it in supported widgets.

## Deleting media

You can delete media from the library.

Important: if a widget is currently using that file, removing it may break that widget's output until you choose a new file.

## Filter tabs

Use filters to find files quickly:
- All
- Images
- Videos
- Audio
