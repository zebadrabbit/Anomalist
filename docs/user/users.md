# Users and Permissions

Anomalist supports three roles so you can share control safely.

## Which role should I use?

- Owner: You. Full control, including user management.
- Editor: Trusted co-host or lead mod who can build and manage overlays.
- Moderator: Day-to-day overlay operators who run approved actions during stream.

## How to create users

1. Open User Management from the gear icon in the dashboard.
2. Click Add User.
3. Enter username, password, and role.
4. Save.

The new user can now sign in from the dashboard URL.

## Per-user permission overrides

You can adjust a specific user's permissions beyond their base role.

In plain terms: if one mod needs extra abilities, you can grant only that person those extra controls without changing everyone else.

How to do it:
1. Open User Management.
2. Find the user.
3. Open Permissions.
4. Set specific permissions to Grant, Deny, or Role Default.

## Security note

Share the dashboard URL only with people you trust.

The URL is not public and requires login, but anyone with valid credentials can control your overlay actions.
